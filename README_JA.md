# Murakami Compressor Suite（日本語）

圧縮をプリセットではなく「探索問題」として解きます。
Probe（短尺/縮小）で多数試し、Finalは最良候補で1回だけ実行します。

動画は SSIMで全候補スクリーニング → 上位だけ VMAF（Top-M）で再評価（可能なら）
で画質劣化を抑えつつサイズを攻めます。
